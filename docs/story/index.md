## What This Isn't

**Not a religion.** No beliefs required. No guru. If anything, this project suggests that "spiritual" teachings might be technical observations that got mystified over time.

**Not a Theory of Everything.** I'm not claiming to have solved reality. I'm pointing at a pattern and asking if you see it too.

**Not finished.** The pattern keeps showing up in new places. The documents keep evolving.

**Not certain.** This is pattern recognition, not proof. I could be seeing faces in clouds. The invitation is to look for yourself.

## Methodology

How this project works:

1. **Notice pattern** — Something shows up in one domain
2. **Test across domains** — Does it appear elsewhere?
3. **Generate expressions** — Create 可s (articulations) from different angles
4. **Trust resonance** — Let the 常 (invariant) reveal itself through the 可s
5. **Revise continuously** — Nothing is final

The collaboration with AI follows the same structure. I bring questions and pattern-hunches. AI helps build tools to test them, articulate what I'm seeing, catch overreach. Every document emerged from that collaboration.
