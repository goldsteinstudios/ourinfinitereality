RSM

Recursive Structural Model
center / periphery — The pattern that describes itself

The RSM is not a theory about reality—it's a description of how description works. A pattern for noticing patterns. It doesn't claim to be true; it claims to be useful. If it stops being useful, discard it.

P₀ → O₁ → G₁ → P₁ → O₂

The Six Guardrails

Constraints that keep the model honest. If you violate these, you've left the RSM.

G1
No privileged frame — No observer position is special
G2
Preserve paradox — Don't resolve it, contain it
G3
Cross-domain — Must apply across domains or it's not the pattern
G4
Scale humility — Claims about "everything" must include this claim
G5
Invitation over assertion — "Look here" not "believe this"
G6
Describe, don't prescribe — What is, not what should be
