# The Twelve Books: Guodian-Based Structural Reconstruction

*A complete character-level analysis of the Dao De Jing, organized by structural function*

---

## Overview

This document compiles all twelve books of the RSM-aligned DDJ reconstruction, based on Guodian manuscript evidence (~300 BCE). Each book groups chapters by structural function rather than traditional numbering, with complete character tables for tracing and calligraphy practice.

**Translation Principle**: All translations use DESCRIPTIVE framing ("X absent") not PRESCRIPTIVE ("don't X"). The DDJ documents what happens, not what you should do.
